## backend 실행

1. build

   ```shell
   cd backend
   gradlew clean build
   ```

2. 실행

   ```shell
   cd build/libs
   java -jar backend-0.0.1-SNAPSHOT.jar
   ```

3. swagger port  
   http://localhost:8080/swagger-ui.html
